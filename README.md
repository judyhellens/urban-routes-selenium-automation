# Urban Routes — Selenium Automation Project

## 📌 Project Overview
Automated testing project developed using Selenium WebDriver, Pytest, and Python to validate the complete taxi ordering flow.

---

## 🚀 Technologies & Tools
- Selenium
- Python
- Pytest
- POM
- ChromeDriver
- Automated Testing
- UI Testing
- GitHub

---

## ✅ Automated Test Scenarios

- Set route address
- Select Comfort plan
- Phone number validation
- SMS code retrieval
- Add payment card
- Add driver comments
- Request blankets and tissues
- Request ice cream
- Complete taxi request flow

---

## 🏗️ Framework Structure

Implemented:
- Page Object Model (POM)
- Setup & Teardown
- Selenium WebDriver
- Pytest execution

- Automated end-to-end flows
