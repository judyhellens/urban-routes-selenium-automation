# Urban Routes — Selenium Automation Project

## 📌 Project Overview
Automated QA testing project developed using Selenium WebDriver, Pytest, and Python to validate the complete taxi ordering workflow in the Urban Routes application.

---

## 🚀 Technologies & Tools
- Selenium WebDriver
- Python
- Pytest
- Page Object Model (POM)
- ChromeDriver
- UI Testing
- Automated Testing
- GitHub

---

## ✅ Automated Test Coverage

The automated tests validate the following user flows:

- Route address selection
- Comfort tariff selection
- Phone number verification
- SMS code confirmation
- Credit card addition
- Driver comments
- Blanket and tissues request
- Ice cream order
- Complete taxi booking workflow

---

## 🏗️ Test Framework Structure

Implemented:
- Page Object Model (POM)
- Setup & Teardown methods
- Selenium WebDriver automation
- Pytest execution framework

---

## ⚡ Key Achievements

- Automated repetitive end-to-end testing scenarios
- Improved test organization and maintainability
- Validated UI functionality and user workflows
- Reduced manual testing efforts through automation

## 📂 Project Files

- `main.py` → Main automation execution
- `pages.py` → Page Object Model structure
- `helpers.py` → Support functions
- `data.py` → Test data
- `test_selenium.py` → Automated test scenarios
- `requirements.txt` → Project dependencies
